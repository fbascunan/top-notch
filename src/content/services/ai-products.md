## Software inteligente, no solo automatizado

Construimos aplicaciones web que integran modelos de lenguaje (LLMs) como componente central del producto — no como un chatbot agregado al final, sino como la experiencia misma. Tutores IA, asistentes de estudio, herramientas de análisis con comprensión de contexto.

### Lo que entregamos

- **Tutores y asistentes IA** — Aplicaciones educativas con tutores conversacionales que adaptan el contenido al nivel del usuario, generan ejercicios y dan retroalimentación en tiempo real.
- **Integración de LLMs en productos** — Conexión con APIs de OpenAI, Claude, MiniMax u otros modelos, con prompt engineering optimizado para cada caso de uso.
- **Backend con Supabase** — Autenticación, base de datos PostgreSQL, almacenamiento de archivos y funciones edge — todo en una plataforma gestionada.
- **Interfaces reactivas** — Frontends modernos con React y TypeScript, diseñados para interacciones conversacionales fluidas.

### Proyectos que lo demuestran

- **[PAES O7](/projects/paes-o7)** — Plataforma de práctica para el examen PAES (admisión universitaria en Chile). Tutor IA con MiniMax que explica soluciones paso a paso, banco de preguntas y seguimiento de progreso. React + TanStack Router + Supabase.

### Stack tecnológico

React, TypeScript, TanStack Router, Supabase, OpenAI, Claude, MiniMax, Vite, PostgreSQL.
