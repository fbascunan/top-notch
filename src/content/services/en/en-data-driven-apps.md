## Open data at the service of people

We develop progressive web applications (PWAs) focused on transparency and public utility. Tools that transform scattered data into accessible, comparable, and useful information for citizens and organizations.

### What we deliver

- **Installable PWAs** — Web applications that work like native apps: install on the device, load fast, and work offline.
- **Crowdsourced data** — Collaborative data collection systems where users contribute and validate data, creating reliable information databases.
- **Data visualization** — Interfaces that present complex data clearly: comparators, rankings, interactive maps, and filterable tables.
- **Public data APIs** — Endpoints that expose structured data for third-party consumption, fostering the open data ecosystem.

### Projects that prove it

- **[Notarías de Chile](/en/projects/notarias-de-chile)** — Civic PWA that exposes and compares notary costs in Chile. User-crowdsourced data, with notary rankings by price and location. TanStack Start + Drizzle ORM + SQLite.

### Tech stack

TanStack Start, React, TypeScript, Drizzle ORM, SQLite, PWA APIs, Tailwind CSS.
