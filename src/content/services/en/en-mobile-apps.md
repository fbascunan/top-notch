## Native apps from a single codebase

We develop cross-platform mobile applications using Expo and React Native. One codebase that runs on both Android and iOS, with native feel, smooth performance, and access to device capabilities.

### What we deliver

- **Cross-platform apps** — One codebase, two platforms. React Native with Expo for rapid development and streamlined deployment to App Store and Google Play.
- **Cloud service integration** — Firebase for authentication, storage, and push notifications. Serverless backend that scales automatically.
- **AI-powered features** — Integration of language models and AI APIs for intelligent in-app features: content generation, conversational assistants, voice processing.
- **Native mobile UX** — Smooth navigation, touch gestures, offline support, and platform-adapted design.

### Projects that prove it

- **[NoT Preocupeit](/en/projects/not-preocupeit)** — Language learning app with a conversational AI tutor. Expo + Firebase + MiniMax for audio and dialogue generation. Android first, with Google Play publication planned.

### Tech stack

React Native, Expo, TypeScript, Firebase, MiniMax, EAS Build, React Navigation.
