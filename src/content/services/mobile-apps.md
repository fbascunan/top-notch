## Apps nativas con un solo código

Desarrollamos aplicaciones móviles multiplataforma usando Expo y React Native. Un solo código base que corre en Android e iOS, con experiencia nativa, rendimiento fluido y acceso a funcionalidades del dispositivo.

### Lo que entregamos

- **Apps multiplataforma** — Un código, dos plataformas. React Native con Expo para desarrollo rápido y despliegue simplificado en App Store y Google Play.
- **Integración con servicios cloud** — Firebase para autenticación, almacenamiento y notificaciones push. Backend serverless que escala automáticamente.
- **Funcionalidades con IA** — Integración de modelos de lenguaje y APIs de IA para funciones inteligentes dentro de la app: generación de contenido, asistentes conversacionales, procesamiento de voz.
- **UX móvil nativa** — Navegación fluida, gestos táctiles, soporte offline y diseño adaptado a cada plataforma.

### Proyectos que lo demuestran

- **[NoT Preocupeit](/projects/not-preocupeit)** — App de aprendizaje de idiomas con tutor IA conversacional. Expo + Firebase + MiniMax para generación de audio y diálogos. Android first, con plan de publicación en Google Play.

### Stack tecnológico

React Native, Expo, TypeScript, Firebase, MiniMax, EAS Build, React Navigation.
