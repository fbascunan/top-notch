## Presencia digital que convierte

Construimos sitios web institucionales y plataformas digitales que representan tu marca con excelencia técnica. Cada proyecto está optimizado para velocidad de carga, posicionamiento en buscadores y experiencia de usuario en todos los dispositivos.

### Lo que entregamos

- **Sitios institucionales** — Sitios web corporativos y de servicios construidos con Astro, optimizados para SEO y rendimiento (Lighthouse 90+).
- **Landing pages de conversión** — Páginas diseñadas para captar leads y convertir visitantes, con formularios integrados y analíticas.
- **Plataformas web con gestión** — Sitios con panel de administración, autenticación y gestión de contenido usando Supabase como backend.
- **Internacionalización** — Sitios multilingüe (ES/EN) con rutas localizadas, SEO por idioma y contenido traducido.

### Proyectos que lo demuestran

- **[ClimaTotal](/projects/climatotal)** — Sitio institucional para empresa de climatización (HVAC). Astro + Tailwind CSS, desplegado en Netlify con Lighthouse perfecto.
- **[TopNotch](/projects/top-notch)** — Este mismo sitio: plataforma web con gestión de proyectos, autenticación OAuth, Server Islands y contenido bilingüe.

### Stack tecnológico

Astro, TypeScript, Tailwind CSS, Supabase, Netlify, Cloudflare, Node.js, PostgreSQL.
